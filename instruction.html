Ввод чисел

  1. Для ввода числа используйте клавиши 0–9 в требуемой последовательности (цифры появляются слева направо).
  2. Для ввода числа в научной нотации:
  • Введите нормализованную мантиссу.
  • Нажмите клавишу для ввода экспоненты.
  • Введите порядок числа (с положительным или отрицательным знаком).
  1. Для ввода числа с плавающей точкой:
  • Введите целую часть.
  • Нажмите клавишу ..
  • Введите дробную часть.
  1. Для ввода числа в смешанном формате:
  • Введите число.
  • Нажмите и задайте порядок.

Если при вводе числа ошибка, нажмите Cx и введите значение заново.

Важное замечание

  1. Ввод данных на калькуляторе блокируется, если мантисса уже содержит 10 цифр. В этом случае нажатие цифровых клавиш не влияет на отображение.
  2. При вводе цифр экспоненты каждая новая цифра занимает младшую значащую позицию, сдвигая старую информацию влево. Это приводит к потере самой значимой цифры экспоненты.

Автоматические операции

Для выполнения операций +, -, ×, ÷, yˣ, sqrt{x} и sqrt{xʸ}​:

  1. Нажмите клавишу C.
  2. Введите первое число.
  3. Нажмите операционную клавишу в соответствии с таблицей 1.
  4. Введите второе число.
  5. Завершите операцию нажатием =.

Для выполнения функций, таких как ln⁡X, sin⁡X, sqrt{X}, и других:

  1. Нажмите клавишу C.
  2. Введите число.
  3. Нажмите соответствующую операционную клавишу в соответствии с таблицей 2.

Примечания:

  1. Если на дисплее появляются десятичные точки во всех позициях, это указывает на переполнение (например, ln⁡(−5) или переполнение памяти стека).
  • Для продолжения вычислений нажмите клавишу C или Cx.
  • После нажатия Cx калькулятор возвращается в состояние, предшествующее операции, вызвавшей переполнение.
  1. Для вычисления тригонометрических функций введите аргумент в радианах.
  2. Перевод градусов в радианы осуществляется по формуле

Использование RAM (оперативной памяти)

  • Калькулятор оснащен двумя регистрами памяти:
  • Первый регистр памяти: адресуется клавишей зап и числовыми клавишами.
  • Второй регистр памяти: адресуется клавишей C(3) 2.

  • Сохранение значения:  — запись числа в первый регистр памяти.
  • Чтение значения:  — чтение числа из второго регистра памяти.
  • Когда используется первый регистр, возможно выполнение операций с постоянными.
